template<class InterfaceType>
class apf::Interface< InterfaceType >

智能指针接口

注解
Interface是一个模板智能指针，内部使用引用计数 初始化完一个Interface后需要检查是否有效，确定有效后才能正常使用
Interface<IExample> example("example_class_id");
if (example) {
example->SayHello();
}
example.Release();
